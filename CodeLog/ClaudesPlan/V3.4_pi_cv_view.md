# V3.4 — PI CV-Style "Read More" Section

## Goal
Transform the PI's "Read More" / expanded section on the Team page into a proper CV-style view with:
1. **Collapsible sections** — each category (Publications, Patents, etc.) is a `<details>` element
2. **Filter bar** — chip-style filter at the top to show All or a specific section
3. **Citation counts** — journal papers display their citation count
4. **Default sort by year** — all items within each section ordered by year descending

## Approach

### 1. Enrich `syncCVToProfile` (user-system.js)
Currently syncs only `{title, url}` per paper. Expand to include:
- **Papers:** `year`, `citations`, `journal`, `authors`, `volume`, `issue`, `pages`
- **Posters (conferences):** `year`, `conference`
- **Presentations:** `year` (extracted from `date`), `event`
- **Patents:** `year` (from `filing_date`), `status`, `inventors`

This data flows: CV Builder → `cvData` (Firestore) → `syncCVToProfile` → `users/{uid}` → team page.

### 2. Rewrite `buildPiExpandedHTML` (app.js)
- Wrap each section in `<details class="team-expanded-section">` (reuse existing CSS)
- Add a filter bar above sections with chips: All | Publications | Patents | Presentations | etc.
- For journal papers: show `(Year)` and citation badge `[Cited: N]`
- For all items: show year in parentheses
- Sort items within each section by `year` descending (fall back to title alpha)
- Filter bar click shows/hides `<details>` elements by data attribute

### 3. CSS additions (styles.css)
- `.pi-cv-filter` — horizontal chip bar for section filtering
- `.pi-cv-citation` — small badge for citation count
- `.pi-cv-year` — muted year display
- `.pi-cv-meta` — journal name + volume/issue/pages line

### Files to modify
- `user-system.js` — `syncCVToProfile()` method
- `app.js` — `buildPiExpandedHTML()` function
- `styles.css` — new PI CV styles

### Key decisions
- Graceful degradation: items without `year` sort to the bottom
- Citation count only shown when > 0
- Filter bar only appears when there are 2+ non-empty sections
- All sections start collapsed except the first non-empty one
