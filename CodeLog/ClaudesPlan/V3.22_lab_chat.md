# V3.22 — Lab Chat

## Goal
Build a real-time chat application integrated into the lab apps ecosystem, inspired by Discord/Slack but tailored for a small research lab.

## Key Features
- Channels with admin-defined categories; anyone can create
- Subscription-based notifications; all channels visible/browseable
- User-organized sidebar (custom groups, drag-to-reorder)
- Direct messages (1:1 and group)
- Threaded replies
- Reactions, @mentions, pinned messages, message edit/delete
- File sharing via Google Drive (shared lab folder)
- Read receipts (per-message "seen by")
- Browser notifications (push notifications deferred to phone app)
- Simple markdown, client-side search

## Approach
- Standard lab app pattern: `apps/chat/` with index.html, app.js, styles.css
- 5 new Firestore collections: chatConfig, chatChannels, chatMessages, chatReadState, chatUserMeta
- Real-time messaging via onSnapshot (max 4 listeners active)
- Google Drive uploads via Google Identity Services OAuth (same pattern as Equipment Scheduler)
- CSS prefix: `chat-`
