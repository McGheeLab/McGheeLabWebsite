# V3.27 — Bug & Quality Fixes

## Goal
Address ~19 bugs and quality issues reported across Chat, Equipment, Huddle, and Mobile apps from `CodeLog/Feature requests/labappfeatures.md`.

## Approach
Prioritized fixes into 4 tiers (critical bugs → UX bugs → quality improvements → investigation) and implemented all actionable items.

## Files Modified
- `apps/chat/app.js` — Fixes 4-9, 11-13, 17-18
- `apps/chat/styles.css` — Fixes 8, 11-14
- `apps/equipment/app.js` — Fixes 1-3, 15
- `apps/huddle/app.js` — Fix 16
- `apps/shared/app-base.css` — Fix 10

## Key Decisions
- **Drafts**: Used localStorage per-channel, debounced save on input
- **Action bar**: Changed from static bar above input to floating bar positioned above selected message
- **Pinned messages**: Converted from modal overlay to inline collapsible panel between header and feed
- **Multi-day drag**: Removed single-column constraint in equipment drag, added cross-column highlighting
- **Huddle drag-to-create**: Added pointer-based drag on empty cells that pre-fills the add plan form
- **Mobile layout**: Apps with internal scroll (chat, equipment, huddle) now use `overflow: hidden` on `#app` with child-managed scroll
- **Notification counts**: Quick fix — return 0 if last message is from current user
- **Long history**: Capped fallback subscription to MSG_PAGE_SIZE most recent messages
