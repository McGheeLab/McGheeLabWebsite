# V3.15 — Grouped Navigation (Desktop Dropdowns + Mobile Tab Bar)

## Goal
Replace the flat hamburger nav with grouped, platform-optimized navigation:
- **Desktop (≥768px):** Horizontal nav with dropdown groups on hover/click
- **Mobile (<768px):** Fixed bottom tab bar with group sheets
- **Mission** is now the splash page — accessed via logo click, not a nav link

## Navigation Groups
| Group | Pages | Desktop | Mobile |
|---|---|---|---|
| Research | Research, Projects, News | Dropdown | Bottom sheet |
| People | Team, Opportunities | Dropdown | Bottom sheet |
| Classes | Classes | Direct link | Direct tab link |
| Other | Lab Apps, Dashboard, Admin, Login | Inline (auth-gated) | "More" bottom sheet |

## Desktop Implementation
- `#desktop-nav` contains `<li class="nav-group">` items with `<button class="nav-group-btn">` + `<ul class="nav-dropdown">`
- Hover opens dropdown, mouseleave closes it. Click toggles for touch-screen desktops.
- Click outside or route change closes all dropdowns.
- Group button gets `.group-active` class when any child route is current.
- Chevron SVG inside button rotates on open.

## Mobile Implementation
- 4 bottom tabs: Research (group), People (group), Classes (direct), More (group)
- Group tabs (`button.tab-group-btn[data-group]`) toggle a `#group-sheet-{name}` bottom sheet
- Each sheet has backdrop dismiss, link-click dismiss, Escape key dismiss
- Only one sheet open at a time (opening one closes others)
- `NAV_GROUPS` map in JS drives group-aware active highlighting

## Files Modified
- `index.html` — Restructured `#desktop-nav` with dropdown groups, new `#bottom-tabs` with group buttons, 3 group sheets (research, people, more)
- `styles.css` — Section 03b: `.nav-group`, `.nav-group-btn`, `.nav-dropdown` dropdown styles. Section 03c: `.group-sheet` bottom sheet styles (replaced `.more-sheet`)
- `app.js` — `setupDesktopDropdowns()`/`closeAllDropdowns()` for desktop, `setupBottomTabs()`/`toggleGroupSheet()`/`closeAllSheets()` for mobile, group-aware `setActiveTopNav()` with `NAV_GROUPS` map
- `user-system.js` — No changes needed (existing IDs still work)

# V3.15 — Navigation Redesign (Desktop Inline + Mobile Bottom Tabs)

## Goal
Replace the hamburger-only navigation with two platform-optimized patterns:
- **Desktop (≥768px):** Horizontal inline nav bar in the top banner — one-click access to all pages.
- **Mobile (<768px):** Fixed bottom tab bar with 4 primary tabs + "More" sheet for remaining pages.

## Approach

### Desktop
- New `#desktop-nav` element inside `.top-banner` with all links as horizontal pills.
- Banner grid changes from `1fr auto auto` (brand, user, hamburger) to `auto 1fr auto` (brand, nav, user).
- Hamburger and drawer hidden via `display: none !important`.
- Active page highlighted with `aria-current="page"` + accent background.

### Mobile
- Hamburger and drawer hidden at `<768px` as well.
- Banner grid simplified to `1fr auto` (brand, user).
- Fixed bottom tab bar (`#bottom-tabs`) with 5 items: Home, Research, Team, News, More.
- "More" button opens a bottom sheet (`#more-sheet`) with: Projects, Classes, Opportunities, auth-gated items, Login.
- Sheet has backdrop dismiss, link-click dismiss, Escape key dismiss.
- Safe area insets for iPhone home indicator.
- Footer gets extra bottom padding so content isn't occluded.

### Tab Selection Rationale
| Tab | Route | Why |
|---|---|---|
| Home | `#/mission` | Landing page, most common entry |
| Research | `#/research` | Core academic content |
| Team | `#/team` | Second most visited |
| News | `#/news` | Engagement driver, frequently updated |
| More | (sheet) | Everything else + auth items |

### Auth-Gated Items
- Present in all 3 nav surfaces (drawer, desktop nav, more sheet) with distinct IDs.
- `user-system.js` `updateNavigation()` toggles all three sets simultaneously.
- IDs: `nav-*` (drawer), `dnav-*` (desktop), `more-*` (sheet).

## Files Modified
- `index.html` — Added `#desktop-nav`, `#bottom-tabs`, `#more-sheet` markup
- `styles.css` — New sections 03b (desktop nav), 03c (bottom tabs + sheet), z-index updates, reduced motion
- `app.js` — `setupBottomTabs()`, `toggleMoreSheet()`, `closeMoreSheet()`, updated `setActiveTopNav()` for all nav surfaces
- `user-system.js` — `updateNavigation()` now toggles items across drawer, desktop nav, and more sheet

## Key Decisions
- Old drawer markup left in DOM but hidden everywhere — enables quick rollback
- `--tab-bar-height: 60px` CSS variable for consistent spacing
- Desktop nav uses `overflow-x: auto` as safety valve if links overflow at 768–900px
- More sheet respects `prefers-reduced-motion: reduce`
