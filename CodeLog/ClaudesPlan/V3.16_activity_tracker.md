# V3.16 — Activity Tracker

## Goal

Add a standalone lab app that lets each team member log daily activities, categorize them into a hierarchical taxonomy (Research, Coursework, Service, etc.), track time, and visualize trends. Includes ML-powered auto-categorization, optional AI integration via Anthropic API, milestone marking for annual reviews, and voice input.

## Approach

- **Standalone app** at `apps/activity-tracker/` — registered in LAB_APPS, accessed via `#/apps/activity-tracker`
- **Privacy**: Owner-only Firestore rules — no admin or other user can see tracker data
- **Auth**: Via AppBridge (embedded postMessage or standalone Firebase)
- **ML**: Multinomial Naive Bayes text classifier (~80 lines vanilla JS), no external dependencies, trains incrementally on each categorization
- **AI**: Per-user Anthropic API key (same pattern as CV builder), batch categorization with human approval
- **Charts**: Chart.js loaded in app's own index.html for analytics dashboard

## Data Model

- `trackerData/{userId}` — categories tree, ML model weights, settings
- `trackerEntries/{userId}/entries/{entryId}` — individual task entries with date, categoryPath, duration, milestone, source

## Key Decisions

1. **Standalone app** (not inline page) for privacy isolation and Lab Apps hub integration
2. **Naive Bayes** over tf.js — lightweight, instant training, small model state
3. **Subcollection** for entries — scales beyond Firestore 1MB doc limit
4. **Owner-only rules** — no admin read access, unlike other collections
5. **5 default top-level categories** with 30+ subcategories covering academic lab work
6. **Duration auto-parsing** from natural language ("90m", "1.5h")
7. **Milestone stars 1-5** for annual review aggregation, not task importance

## Sections Implemented

- **Daily**: Date nav, task input (text + voice), task list with category assignment, ML suggestions, AI batch categorize, star ratings, daily summary chips, unlogged calendar event prompts
- **Weekly**: 7-day grid with task previews, weekly totals
- **Analytics**: Metrics (total time, days logged, avg/day, tasks, milestones), Chart.js charts (doughnut, line, horizontal bar, milestone list), AI insights
- **Calendar**: Google Calendar OAuth2 connection (via Google Identity Services), day event list with import/skip buttons, "Import All Unlogged" batch import, auto-detect token expiry, date navigation
- **Categories**: Tree view with add/remove subcategories at any level
- **Settings**: API key status, ML model info/reset, working hours, Google Calendar client ID, JSON export
